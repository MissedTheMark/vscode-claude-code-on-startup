# Claude Code on Startup

Automatically opens a Claude Code tab when VSCode starts. Choose between resuming your last session or opening a fresh tab each time.

## Settings

| Setting | Options | Default | Description |
|---------|---------|---------|-------------|
| `claudeCodeOnStartup.startupBehavior` | `openLast`, `openNew` | `openLast` | Whether to reopen the last Claude Code session or always open a fresh tab |
